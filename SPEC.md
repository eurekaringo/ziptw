# 郵遞區號查詢系統 - 規格說明

## 主要功能
1. 郵遞區號即時搜尋建議
2. 響應式設計，支援手機與桌面
3. 多語系切換（預設：繁體中文、英文）
4. 一鍵複製郵遞區號
5. 隱私權政策頁面

## API 規格
- 查詢 API: `/api/zipcode?query=關鍵字`
  - 輸入：`query`（字串）
  - 回傳：符合的郵遞區號與地址陣列

## UI/UX
- 搜尋框置中
- 搜尋結果下拉顯示
- 支援鍵盤上下選擇建議
- 手機版自動放大輸入區

## 測試規格
- 搜尋「台北」應出現所有台北市的郵遞區號
- 輸入不存在的地名，顯示「查無結果」
- 點擊複製按鈕，內容正確複製到剪貼簿

## 其他
- 程式碼需通過 ESLint、Prettier 格式檢查
- 需有基本單元測試（Jest）
